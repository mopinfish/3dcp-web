3D Cultural Properties Project

文化財を3Dで体験する
新しいデジタル文化遺産プラットフォーム

地域の貴重な文化財をオープンデータと3D技術で可視化し、次世代に継承していくプロジェクト。誰でも参加できるユーザー参加型の文化遺産プラットフォームです。

🚀 主な機能

🗺️

インタラクティブマップ表示

MapLibre GLを活用した高性能な地図上に文化財を表示。地理的位置関係を直感的に把握でき、効率的な文化財巡りが可能です。

🎨

3Dモデル閲覧機能

Three.jsとLuma.ai技術による高品質な3Dモデルをブラウザで直接閲覧。文化財を360度から詳細に観察できます。

📊

オープンデータ活用

政府・自治体のオープンデータと連携し、文化財情報を自動収集・更新。常に最新の情報を提供します。

👥

ユーザー参加型プラットフォーム

誰でも文化財の3Dモデルを作成・共有可能。コミュニティ主導で文化財データベースを拡充していきます。

🌐 最新技術の活用

フロントエンド: Next.js (App Router) + React + Three.js

バックエンド: Django + GeoDjango + PostgreSQL

3D技術: Luma.ai + WebGL

地図機能: MapLibre GL + OpenStreetMap
